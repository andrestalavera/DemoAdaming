# Rappels

## Operateurs C#

Assigner une valeur à travers un opérateur :

| Raccourci | Equivalent de... | Description |
| --- | --- | --- |
| x++ | x = x + 1 | Additionne 1 à `x` |
| x-- | x = x - 1 | Soustrait 1 à `x` |
| x += y | x = x + y | Additionne `y` à `x` |
| x -= y | x = x - y | Soustrait `y` à `x` |
